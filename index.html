Java Utilities

Download the latest version of these utilities including source code. (Jar - 85k)

[Browse the source code. | View the JavaDoc.]

  • Base64 - Encode and decode base 64 in Java.
  • Browser - Open a browser from your Java application on Windows, Unix, or Macintosh.
  • CSV Parser - Java library to read values from a file that is in Comma Separated Value (CSV) format.
  • Password Dialog - A password dialog box written for Java Swing.
  • String Tokenizer - A StringTokenizer replacement with enhancements.
  • Straight Stream Reader - An input stream reader that does not character class translation.
