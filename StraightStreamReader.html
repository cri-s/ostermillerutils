Straight Stream Reader

An input stream reader that does not character class translation. You might find this class useful for reading binary files using an API that only support Reader and not InputStreams. One such example is the lexical analyzer JFlex. There is some discussion in its manual about how a class like this might come in handy. This class is included with the binary scanning example that comes with JFlex.
