Line Ending Conversion

Easily converts text files so that they can be read by editors on various systems. Text editors often do not deal with line endings properly for files created on other systems. Squares that appear in Notepad where it should wrap to a new line are often a result of this. This utility does not care what format the file is in, it can read it. It has options to re-write the file in various formats. By default, it will not touch files that it finds to be in binary format, preventing data corruption. The default option is to convert the text file to your system's format.

Text formats supported:

  • DOS
  • Windows
  • UNIX
  • Java
  • Mac

To run the program use the following command line:
java -classpath utils.jar com.Ostermiller.util.LineEnds --help
